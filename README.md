# Projet Les Voisins de Panier - ESGI

## 📋 Description

Ce projet est la mise en place d'une plateforme marketplace de mise en relation des producteurs et consommateurs de produits locaux éco-responsables en circuits courts à Grenoble et dans son agglomération.
Il a été créé dans le cadre du hackathon SEO/GEO/Analytics en M2 Ingénierie du Web à l'ESGI Grenoble.

Ce projet a été réalisé par le groupe constitué de Baptiste SAUVAGE et Nicolas OSBORNE

### 🛠️ Technologies & librairies utilisées

- **Next.js**
- **Piwik PRO**
- **Python**
- **Vercel**

## 📋 Prérequis

Pour exécuter ce projet, assurez-vous d'avoir déjà installé les outils suivants :

- Node.js

## 🚀 Guide d'installation

Pour lancer l'application complète, ouvrez un terminal et exécutez les commandes suivantes :

### 1️⃣ Cloner le dépôt

```bash
- git clone https://github.com/baptwild/hackaton-producteur-locaux.git
- cd hackaton-producteur-locaux

```

### 2️⃣ Installer les dépendances :

```bash
- npm install
```

### 3️⃣ Lancer le projet :

```bash
- npm run dev
```

## 🌐 Accéder à l'application

L'application sera accessible à l'URL : http://localhost:3000
Elle est aussi déployée à l'aide de Vercel et disponible à l'URL: https://hackaton-producteur-locaux.vercel.app/
